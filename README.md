🌦️ WEATHER APP DASHBOARD
A clean and simple weather web application that shows real-time weather data with beautiful nature video backgrounds.

Features:
1.Real-Time Data: Integrates with the OpenWeatherMap API to provide live updates on temperature, humidity, and wind speed.
5-Day Forecast: Extended weather outlook with accurate daily projections.
Glassmorphism Design: A modern "frosted glass" aesthetic using advanced CSS backdrop-filter techniques.
Responsive Layout: Fully optimized for mobile, tablet, and desktop viewing.
High-quality nature video background

Tech Stack:
Frontend: HTML5, CSS3 (Flexbox/Grid), JavaScript 
API: OpenWeatherMap API
Tools: Git/GitHub for version control

Lessons Learned:
How to handle API authentication and fetch data asynchronously.
Implementing glassmorphism effects using CSS back-drop filters.
Managing video assets within a web application for smooth performance.
